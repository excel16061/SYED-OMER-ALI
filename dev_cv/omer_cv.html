Human Activity Recognition (HAR)

Hi, I'm Syed Omer Ali, a developer and researcher working on Human Activity Recognition using deep learning. This page organizes all sections of my HAR project documentation — from Abstract to Source Code — for easy academic review.

Paper Overview
• Project: Human Activity Recognition (HAR)
• Approach: CNN + Transfer Learning
• Dataset: Weizmann Dataset
• Best Model: VGG-16 (~96.95% accuracy)
• Use Cases: Elderly care, security, smart monitoring
Technical Stack
• Python, TensorFlow/Keras
• CNN architecture experimentation
• Feature extraction via transfer learning
• Data preprocessing + augmentation
• Modular documentation
Key Insight
Deep CNNs outperform traditional ML
when combined with transfer learning,
even on small datasets like Weizmann.
