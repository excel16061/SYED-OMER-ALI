Human Activity Recognition (HAR)

Hi, I'm Syed Omer Ali, a developer and researcher working on Human Activity Recognition using deep learning. This page organizes all sections of my HAR project documentation — from Abstract to Source Code — so faculty, reviewers, and collaborators can easily read everything online.

Paper Overview
• Project: Vision-based Human Activity Recognition (HAR)
• Approach: CNN models with Transfer Learning
• Dataset: Weizmann Human Action Dataset
• Best Model: VGG-16 with ~96.95% accuracy
• Applications: Elderly care, surveillance, smart environments

Technical Stack
• Python, CNNs, Transfer Learning
• Dataset preprocessing and augmentation
• Model training and evaluation
• Modular documentation: Intro, Lit Survey, Modules, Test Cases, Source Code

Key Insight
Deep CNNs with transfer learning significantly improve HAR accuracy
over traditional handcrafted features, especially on constrained datasets.
